## Volta
- pin dependency version more easily
```sh
curl https://get.volta.sh | bash
```
- pin the latest version
```sh
volta pin node yarn
```

## Add `.gitignore` using `npx`
```sh
npx gitignore node
```

## Better than cat
- [bat](https://github.com/sharkdp/bat)
```
bat README.md
```

```sh
yarn tsc --init # this will always work
```
